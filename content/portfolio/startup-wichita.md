Startup Wichita is a community organization dedicated to fostering entrepreneurship and innovation in Wichita, Kansas.

We built a modern web platform using Node.js to help connect entrepreneurs, provide resources, and showcase the vibrant startup ecosystem in Wichita. The platform serves as a central hub for the local startup community.
